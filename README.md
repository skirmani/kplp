# Kirmani Partners LP - Investment Pitchbook

[\![GitHub Pages](https://img.shields.io/badge/Live-GitHub%20Pages-00ff88)](https://skirmani.github.io/kplp/)

Interactive single-page investment pitchbook for Kirmani Partners LP. 9-slide presentation with animated charts, performance analytics, and strategy overview.

## Live Demo

**[View the Pitchbook](https://skirmani.github.io/kplp/)**

## Features

- **9 Interactive Slides** - Strategy overview, performance, risk metrics, team, and terms
- **Animated Charts** - Portfolio growth and performance visualization via Chart.js
- **Responsive Design** - Mobile-friendly with keyboard and swipe navigation
- **Dark Theme** - Professional dark UI with accent lighting

## Tech Stack

HTML | CSS | JavaScript | Chart.js | Luxon | GitHub Pages
